# eight_ball_app

## Steps:
### Install ruby
* $ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
### Install rails
* $ gem install rails -v 4.2.6
### Install postgresql
* $ brew install postgresql
### Update dependencies
* $ bundle install
### Run rails server
* $ rails s
### Open app by going to localhost:3000 in browser
